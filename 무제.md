## 챗봇 대화 흐름 개선 요약

### 1. 여러 업무를 전부 파악하도록 변경

- **원래**: 멤버가 여러 업무를 말해도 **하나만 골라서** 후속질문하고 나머지는 무시
- **개선**: 멤버가 말한 **모든 업무에 대해 순차적으로** 질문. 업무 A 질문 끝나면 → 업무 B → 업무 C 순서로 진행

### 2. 하나만 말하면 다른 업무도 있는지 확인

- **원래**: 멤버가 업무 1개만 말하면 바로 그 업무에 대해서만 질문 시작
- **개선**: "혹시 이 외에 진행 중인 다른 업무도 있으실까요?" 라고 먼저 확인한 뒤 진행

### 3. 멤버가 질문을 거부하면 바로 전환

- **원래**: 거부에 대해 장황하게 이유를 설명하고 같은 방향으로 다시 질문
- **개선**: 거부 신호가 오면 **설명 없이 해당 주제를 즉시 놓고**, 다른 업무로 넘어가거나 요약 단계로 전환

### 4. 인사말에서 여러 업무를 말하도록 유도

- **원래**: "가장 중요하게 다루고 계신 업무는 무엇인가요?" → 하나만 말하게 유도
- **개선**: "진행하고 계신 업무들을 편하게 알려주세요!" → 여러 개를 자연스럽게 나열하도록 유도

### 5. 질문 수 안내를 유연하게

- **원래**: "3-4가지 질문을 드릴 예정입니다" → 실제로는 업무 수에 따라 더 많은 질문이 나감
- **개선**: "각 업무에 대해 간단한 질문들을 드릴 예정입니다"
