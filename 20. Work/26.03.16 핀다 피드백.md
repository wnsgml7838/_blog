
# 2시 허규범님 
오블릿 사용하 기전에는?
- 개인적으로 주간단위로 진행을 했었음
- 1주에 한번씩 총 4명을 스케줄링해서
- 구글 닥스로 공유하면서
- 질문유형(워크세션, 커리어세션, 하고싶은말)로 정형화된 내용으로 생성해서 공유하면 각자 내용 작성해서 1on1진행
- 매번 문서를 생성하고, 진행하고, 그 이후에 특이사항이나 이런것들은 개별적으로 기록을 했었음
왜 이거를 사용하고 신경쓰고 사용을 했는지?
- 워크세션
	- 데일리 스크럼을 진행하긴함 
		- 개인별로 체크하고, 피드백을 주기에는 데일리 스크럼은 시간이 부족함
	- 주간단위로 업무 보고 및 공유, 피드백 정도로 채택을 해서 심도 있게 주 1회 진행
	- 세밀하게 인지하고 피드백을 줌으로써 놓치는 부분 없어 개발적인, 팀적으로 알 수 있었음
- 커리어세션
	- 업무 외적으로 커리어를 어떻게 발전시키는가에 대해서 같이 고민하고 도움도 주는(리드로서의 중요한 역할이라고 생각해서)
	- 어떤 고민이 있고, 권한, 팀, 어떻게  도움을 줄지 등등 
구성원의 반응은?
- 이전 사람이 1on1했어서 2명정도는 익숙했음
- 그래도 잘 활용을 해주는 것 같다
내부적으로는 어떤가?
- 위클리 단위로 하는 사람은 많이는 없는거같다
- 2주나 1달주기는 있는것 같음 
- 필요성이나 니즈는 강한데, 시간분배 등을 통해서 많지는 않음
오블릿 사용 이전에 어떤게 번거로웠나
- 40~50 정도 준비하는 시간이 걸림
	- 구성원마다 다르지만 적게는 30분
	- 처음보다는 시간이 좀 단축됐음

오블릿 사용 이후 어떤게 달라졌나?
- 스케줄링 -> AI 질문 생성 : 이전보다 훨씬 더 짧아짐 
- 미팅 이후에 요약이나 분석해주는 거 보거나 1/5 수준으로 짧아짐 
- 분석중에 발화 비율에 대해서 (알려주는거)에 대해서 좋았음 
	- (구체적인 피드백이라고는 안함 / 보여주는 것에 대한 내용이였음)
오블릿 사용의 팀원의 반응은?
- 요약, 분석 내용을 좀 보는것같다 팀원이
	- 다음주에 해야할것, 이번주에 해야할 것 등
	- 미팅 이후에 정보나 자료를 확인하는 것 같다 
- 거부감 관련 해서는?
	- 처음에는 반응이 있지는 않았음
	- 진행을 하면서 이후에 어떤 내용으로 말을 했는지 바로 볼 수 있는 것들이 좋은 반응 
가장 만족스러운거는?
- 요약
	- 요약 내용 보고 마침
- 분석 파트 
	- 끝나면 발화 비율 좀 보고
	- 피드백 부분 보고

발화 비율 관련
- 가끔 9:1이 나와서 어떤 기준으로 발화 비율이 어떻게 나오는지 모르겠다
- 인식률이 가끔 인식이 단어나 이런것들이 부정확해서 수정하거나 이런게 있다
- 녹음이 조금 더 잘 되었으면 좋겠다 
노트북이 가끔 off가 되서 저장된 정보가 가끔 없을때가 있음 
- 특정인이랑 특정 팀원이랑 했을때 3~4번만 있었던것같음
- 플랫폼 문제가 아닌 마이크 off가 안되어서 이거를 사전에 인지할 수 있는 기능이 있으면 좋겠다

분석 하고나서 결과를 볼 때 정확도가 체감상 어느정도인가?
- 했던것에 대해서 잘 이야기를 해주는 것 같음 -> 피드백 부분 
	- 수긍이 되고, 
	- 잘한부분 부족한 부분을 확인할 수 있어서 좋았음
	- 다만 하면 할수록 분석 내용이 비슷한 경우도 많은 것 같음
		- 지난주나 이번주에 대해서 비슷한 내용
		- 다른 사람이랑 했을 때도 
		- 좀 더 다양한 분석 내용이 있으면 좋을 것 같았음
AI 질문 추천을 잘 사용하는지?
- 그전에는 동일한 질문으로 계속 사용했는데, 
- AI가 이전 미팅의 맥락을 잘 반영해서 잘 뽑아주는 것 같아서
	- 맥락이나 내용은 잘 납득이 되서 잘 사용중
	- STT 오전사로 인한 단어 등을 수정하는 정도이다 

미팅 프렙에 대한 경험 
- 이 기능에 대해서는 잘 몰랐음 
- 미팅 프렙과 질문생성과의 차이를 물어봤음(허규범)
질문 생성 퀄리티 
- 지난 미팅에 대한 아젠다난 맥락은 잘 반영하는 것 같음
- 연속성이 좋음
- 다만, 분석할 때 아이스브레이킹 식으로 잘 지냈는지 현황으로 지적을 받는데
	- 어색하긴 한데, 억지로 넣기도 하고 있다
IC 리포트 사용 경험
- 아직 사용은 안해봤음 
나인 박스/대시보드 사용 경험
- 활용 방법을 잘 모르겠어서 사용은 안해봤음
왜 이걸 계속 써야하나?
- 일정관리, 진행하는데에 있어서 리소스 절약이 가능했다
- 분석(피드백)/요약 을 가지고 참고해서 어떻게 개선하고 진행을 할 수 있을지에 대해서 도움을 받았다
